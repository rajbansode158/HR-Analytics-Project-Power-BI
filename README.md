# HR Analytics Dashboard | Power BI

This project is an **HR Analytics Dashboard** created using **Microsoft Power BI** to analyze employee data and generate insights on **attrition, headcount, and workforce trends**.

---

## Key Insights
- Employee & Attrition Analysis  
- Department-wise Trends  
- Gender & Age Distribution  
- Interactive Dashboard  

---

## Tools Used
- Power BI  
- Power Query  
- DAX
